# Bite Buddy Landing Site

Marketing landing page and privacy policy for [Bite Buddy](https://stevendiamante.github.io/bitebuddy-site), an ADHD-friendly meal companion app.

## Local Development

Open `index.html` in a browser. No build step required.

## Image Rebuilds

Requires [cwebp](https://developers.google.com/speed/webp/docs/cwebp) to convert source PNGs to WebP.

## Deployment

Served via GitHub Pages from the `main` branch root.
